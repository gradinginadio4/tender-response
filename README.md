# Homieux Media — B2B Insight Tools

Five simple web applications for Belgian equipment distribution companies.

## Brand Compliance
- Typography: Epilogue (Google Fonts)
- Colors: #000887 (blue), #ED51C2 (pink accent), #FFFFFF (white)
- Clean, minimal, professional aesthetic

## The Five Tools

### 1. Revenue Timing
Insight into seasonal revenue patterns common in equipment distribution.
- URL: `https://[username].github.io/homieux-revenue-timing/`
- 2 screens: Pattern observation + examples
- Soft CTA to Calendly

### 2. Sales Time
Simple calculator showing time allocation patterns.
- URL: `https://[username].github.io/homieux-sales-time/`
- 2 screens: Input + examples
- Interactive slider (no quiz)

### 3. Deal Timing
Observation on deal progression and follow-up approaches.
- URL: `https://[username].github.io/homieux-deal-timing/`
- 2 screens: Pattern + examples
- No assessments or judgments

### 4. Margin Models
Overview of three business model evolutions.
- URL: `https://[username].github.io/homieux-margin-models/`
- 2 screens: Context + three models
- No requirements or gap analysis

### 5. Tender Response
Insight on response time optimization.
- URL: `https://[username].github.io/homieux-tender-response/`
- 2 screens: Observation + examples
- No competitive fear framing

## Common Elements (All Apps)
- Calendly link: `https://calendly.com/homieuxmedia/discovery-call-gratuit-discutons-de-vos-defis`
- Contact: `info@homieuxmedia.com`

## Deployment
1. Create separate repository for each app
2. Upload `index.html` as sole file
3. Enable GitHub Pages (Settings → Pages → main branch)
4. Test URL with `?company=Example` parameter for personalization
